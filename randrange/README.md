# randrange

prints a random value using the specified parameters

## usage

```
usage: randrange.py [-h] [-c COUNT] low high

Print a random value using the specified parameters. The low and high arguments must not be the same. The order of the low and
high arguments is not relevant.

positional arguments:
  low                   the lower boundary for the allowable range
  high                  the upper boundary for the allowable range

optional arguments:
  -h, --help            show this help message and exit
  -c COUNT, --count COUNT
                        the number of random values to print
```
